## 📝 配置清单

### 需要配置的项目

#### 1. GitHub Secrets（必需）
```
GOOGLE_API_KEY      - Gemini API 密钥
EMAIL_USER          - 发件邮箱
EMAIL_PASSWORD      - 邮箱授权码
```

#### 2. config.yaml（必需修改）
```yaml
email_settings:
  receiver: "your_email@example.com"  # 改为你的邮箱
  smtp_host: "smtp.qq.com"            # 根据邮箱类型修改
```

#### 3. 可选配置
```yaml
crawler_settings:
  max_workers: 5                      # 并发线程数
  rate_limit_seconds: 0.5             # 请求间隔
  content_freshness_hours: 48         # 内容新鲜度

feeds:                                 # 可添加/删除 RSS 源
  - name: "自定义源"
    url: "https://..."
    max_items: 5
```
